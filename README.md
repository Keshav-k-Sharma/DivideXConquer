# DivideXConquer# ⚔️ DivideXConquer

**DivideXConquer** is an interactive web-based algorithm visualizer designed to demystify the "Divide and Conquer" paradigm. This tool provides a step-by-step visual representation of how complex problems are recursively broken down into manageable sub-problems, solved, and seamlessly combined.

## 🧠 About the Project
Divide and Conquer is a fundamental algorithm design paradigm. While powerful, it can be difficult for students and developers to visualize recursive calls and the "merging" phase. 

**DivideXConquer** bridges this gap by providing a clean, intuitive interface where users can watch data transform in real-time. It’s built for learners who want to see the "why" and "how" behind the code.

## ✨ Key Features
* **Real-Time Animation:** Watch elements move and change state as the algorithm processes them.
* **Step-by-Step Control:** Pause, play, or adjust the speed of the visualization to match your learning pace.
* **Custom Inputs:** Input your own arrays or data sets to test edge cases.
* **Complexity Metrics:** View Time Complexity $O(n \log n)$ and Space Complexity calculations on the fly.
* **Responsive Design:** Fully optimized for desktop and mobile viewing.

## 🔢 Algorithms Visualized
The platform focuses on classic Divide and Conquer implementations:
* **Merge Sort:** Visualizing the recursive splitting of data into single-element units and the subsequent sorted merging process.
* **Quick Sort:** Highlighting pivot selection, the partitioning logic, and how the array is sorted in-place without extra memory.
* **Matrix Multiplication (D&C):** Demonstrating the standard recursive approach by partitioning large matrices into four sub-matrices for processing.
* **Strassen's Multiplication:** Visualizing an optimized matrix multiplication that reduces recursive calls from 8 to 7 using algebraic identities.
* **Min and Max Finding:** Showcasing how to find the minimum and maximum elements in an array simultaneously by splitting the search space.
* **Largest Subarray Sum:** Implementing Kadane’s logic via Divide and Conquer to find the contiguous subarray with the largest sum.
* **Closest Pair of Points:** Solving geometric problems efficiently by splitting planes and checking for the smallest distance across a dividing "strip."
* **Convex Hull:** Visualizing the "Divide and Conquer" approach to finding the smallest convex polygon that encloses a set of points.
